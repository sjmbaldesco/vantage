# Vantage — site starter

Astro 7 + Tailwind CSS v4 starter implementing the "signal teal" brand system.

## Quick start

```
npm install
npm run dev
```

Open the printed local URL to see the feed and the sample explainer page.

## Where things are

- `src/components/Header.astro` — logo lockup
- `src/components/FactPost.astro` — the feed card that enforces the one-highlight-per-post rule
- `src/pages/index.astro` — the feed
- `src/pages/explained/rice-prices.astro` — sample long-form explainer template
- `src/styles/global.css` — brand color and font tokens (Tailwind v4 CSS-first config)

See `CLAUDE.md` for the full brand system, voice rules, and suggested next steps.
